## 一、linux vi编辑器的使用
[1.菜鸟教程VIM](https://www.runoob.com/linux/linux-vim.html)
常用VIM编辑操作：
1.gg：跳转至文件首行
2.G：跳转至文件尾行
3.H：跳转至当前页首行
4.M：跳转至当前页中间
5.nyy：n为数字，不带数字默认为1，yy为复制
6.p：粘贴
7.nx/X：向后或向前删除字符
8.ndd/D：剪切n行/剪切后留行
